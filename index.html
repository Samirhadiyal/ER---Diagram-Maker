Connection Mode: Inactive

How to Create Connections:

  1. Step 1: Click "Toggle Connection Mode" button or select "Arrow Mode" from dropdown
  2. Step 2: While in connection mode, click and drag from one entity to another
  3. Step 3: Release the mouse to complete the connection with an arrow
  4. Step 4: Click "Toggle Connection Mode" again to exit connection mode

Tip: You can create multiple connections before exiting connection mode
